# akashic-client

> [!NOTE]
>
> 今は単体で動きますが、最終的には [client](../client) が参照するコンポーネントライブラリとなります
>
> - `app` 以下 → React Commonent としてディストリビュート
> - `pages/api` 以下 → Next.js API Handler の実装としてディストリビュート
>   単独で実行可能にしておいたほうがデバッグ用に役立つと判断し、本サブリポジトリは Next.js プロジェクトのままにし、 npm ライブラリ用にビルドするビルドファイルを別途作る。
>   - `/api/content/[id]` の実装時は baseUrl の解決が必要なので引数として設定させるようにする

> [!IMPORTANT]
>
> ニコ生ゲームではスキップ中の画面操作を禁止しているが Akashic 上は禁止していない
> 別途禁止処理を作る必要がある
