# manager-server

サービスのメンテナンス機能を制御するサーバーです。
常時起動させておく必要はありません。

## ドレイン

エンドポイント: `/drain`

稼働中の機能を抑止することで、サーバーを停止してもユーザー影響がない状態にさせる機能です。
ドレイン中は下記の操作を受け付けません。

- 部屋作成
- 部屋延長
- コンテンツ投稿
- コンテンツ更新
- ゲーム削除

## ログの削除

エンドポイント: `/content-logs/delete`

一定期間経過したContent Log, Client Log を削除します

- retetionDays: 保持日数
- includeErrored: エラーログも消すか
