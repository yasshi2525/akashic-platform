# About

Akashic Engine のゲーム実行に必要なサーバー実装です。

すべてのプレイのアクティブインスタンスを実行する専用サーバーです。

## NOTE

> [!NOTE]
>
> 現状、1アクティブインスタンスを生成するごとに Socket.IO で通信路を確保しているため、その分 `akashic-storage` のポートを消費する。1個にまとめることも可能だが、 `akashic-storage` は全プレイヤーからの通信路も保持するため、省リソース効果が低い。もしまとめるなら `schema/amflow` に playId を含ませる変更が必要になる。不必要に複雑化させないために、あえて現状の仕様としている
